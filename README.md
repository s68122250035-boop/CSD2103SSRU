# CSD2103SSRU
ชื่อโปแกรม
SearchScore.java
---------------------------------------------------
Input
รับคะแนนนักศึกษา 10 คน
รับคะแนนที่ต้องการค้นหา 
ตัวอย่าง
70
80
65
90
75
60
55
88
91
72

คะแนนที่ต้องการค้นหา : 90

_______________________________________________________
Process
-สร้าง Array สำหรับเก็บคะแนนนักศึกษา 10 คน
-รับคะแนนจากผู้ใช้และเก็บลงใน Array
-รับคะแนนที่ต้องการค้นหา
-วนลูปตรวจสอบว่าคะแนนที่ค้นหามีอยู่ใน Array หรือไม่
-ถ้าพบแสดง "Found"
-ถ้าไม่พบแสดง "Not Found"
________________________________________________________
Output
ถ้าพบ
Found
ถ้าไม่พบ
Not Found
________________________________________________________
วิธีรันโปรแกรม
1.เปิดไฟล์ SearchScore.java
2.กด Run หรือ Compile โปรแกรม
3.กรอกคะแนนนักศึกษา 10 คน
4.กรอกคะแนนที่ต้องการค้นหา
5.โปรแกรมจะแสดงผลว่า Found หรือ Not Found
__________________________________________________________
สิ่งที่ได้เรียนรู้
การสร้างและใช้งาน Array
การรับค่าด้วย Scanner
การใช้คำสั่ง for ในการวนลูป
การใช้คำสั่ง if-else
การค้นหาข้อมูลใน Array
การใช้ตัวแปรชนิด boolean
