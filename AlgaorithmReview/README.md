## Case Study 1 : Array

คะแนนนักศึกษา

java
int[] scores = {6,8,4,9,7,5,10,3,8,2};

ผลลัพธ์

คะแนนรวม = 62
คะแนนเฉลี่ย = 6.2
คะแนนสูงสุด = 10
คะแนนต่ำสุด = 2
นักศึกษาที่ได้คะแนนตั้งแต่ 7 คะแนนขึ้นไป = 5
นักศึกษาที่ควรทบทวน = คนที่ 3, 8 และ 10

## Case Study 3 : Queue

คิวเริ่มต้น

[P001, P002, P003, P004, P005]

P001
P002

เพิ่ม

P006
P007

ผลลัพธ์

ผู้ป่วยคนถัดไป = P003
